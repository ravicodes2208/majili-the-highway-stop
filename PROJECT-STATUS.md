## Key Decision Log

| Decision | Choice | Rationale |
|----------|--------|-----------|
| Meals? | NO | Lower risk, simpler ops, higher margins, fewer staff |
| Franchise? | NO (own brand) | Highway customers don't care about brand names; save Rs 10-40L |
| AC room? | NO | Open-air with fans, shed roof |
| Roof type | Shed (terracotta color-coated) | Rs 1-1.5L cheaper than RCC, 6 weeks faster, highway-standard look |
| Toilet config | Separate block (3T + 5U) | Doesn't eat indoor space, adequate for highway traffic |
| Outdoor seating | Open sky + umbrellas + shade trees | Phase later if needed, saves Rs 2-3L on covered canopy |
| Coffee machine | Traditional boiler (not espresso) | Cheaper CapEx, same electricity, authentic filter coffee positioning |
| Ice machine | Deferred (Phase 2) | Use freezer + trays initially, add when cold bev volume justifies |

## Location Intelligence

### Why This Spot
1. **Pre-expressway "last stop"**: 10 km before Devarapalli Expressway entry (162 km access-controlled, limited stops)
2. **Petrol bunk adjacent**: Under construction next door; 25-40% of pump customers will spillover
3. **Proven corridor**: 155 bills/day verified at upstream dhaba
4. **No direct coffee competitor**: Only meal-based dhabas exist on NH-365BB
5. **Kusumanchi temple nearby**: Kakatiya-era Shiva temple adds pilgrim baseline traffic
6. **Friend's land**: Favorable lease terms, CA handles financials

### Constraints
- **Eastbound only**: No median crossing, no U-turn access for westbound traffic
- **Expressway uncertainty**: NH-365BG opening changes traffic patterns (expected to BENEFIT, not hurt)
- **New highway**: NH-365BB only opened March 2025, ecosystem still forming
- **Remote location**: Power cuts possible, need inverter + generator backup

### Traffic Sources (Eastbound = Hyd -> Vizag Direction)
1. **Hyderabad departures** -> 2.5 hrs to reach location (primary peak: 8-11 AM arrivals)
2. **Rajahmundry departures** (via expressway) -> 4.7 hrs (primary source for afternoon/evening)
3. **Khammam-bound local traffic** (additional, not significant)
4. **Petrol bunk spillover** (60-160 customers/day potential)

### Four Peak Windows (Coffee/Snacks Model)
1. Morning Coffee: 8-11 AM (Hyd 5-8 AM departures)
2. Midday: 12-3 PM (Vizag travelers arriving)
3. Tea Time: 4-7 PM (HIGHEST revenue window)
4. Evening: 7-10 PM (truckers, night buses)

## Brand Identity

### Theme: Kakatiya Heritage
- **Why**: Kakatiya dynasty (1163-1323 CE) ruled both Telangana AND Andhra Pradesh
- Politically neutral, culturally rich, shared heritage
- Kusumanchi temple on the corridor IS a Kakatiya-era construction
- Visual vocabulary: Kala Thoranam arches, Purna Ghatak, lotus carvings

### Visual Direction
- **Colors**: Forest green, sandstone red, temple gold, lime-wash cream
- **Logo concept**: Coffee cup inside a Kala Thoranam arch
- **Murals**: Monochrome Cheriyal scroll paintings (terracotta on cream)
- **Interior**: Earthy, warm, traditional without being kitschy

### Mural Plan (3-4 walls)
- Entrance: Village hospitality scene (woman offering food, man with instrument)
- Main dining: Kakatiya temple scene
- Behind counter: Traveler's scene (pilgrims on road with rest huts)
- Kids corner: Playful mythology (Krishna, Hanuman)
- Artists: Cheriyal Nakashi family (Siddipet district, 2.5 hrs from Hyderabad)
- Budget: Rs 1.5-2.5L for 3-4 feature walls

### Signage System (2 highway boards initially)
1. **500m advance**: "COFFEE * TOILET * PARKING - 500m" (retroreflective)
2. **100m turn-here**: Brand name + logo + "ENTER HERE" arrow
3. (Phase 2: 800m awareness board)
